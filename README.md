🍽️ Catering Services Website


A multi-page catering service website designed to simulate a real-world food ordering platform. The project focuses on frontend design, UI planning, and creating a smooth user experience for browsing and interacting with catering services.

Features

* Multi-page website (Home, Menu, Cart, About Us)
* Structured navigation and user flow
* Menu browsing and basic interaction
* Clean and responsive UI design

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

 My Contribution
